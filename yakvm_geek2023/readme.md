# README 

为了加速安全产品和安全工具的工程化研发，我们创建了一门新的语言（Yaklang），并且同时实现了一个栈虚拟机（YakVM）。
Yaklang 是一门图灵完备的编程语言，其语法由上下文无关文法定义。它运行在 YakVM 上。
Yaklang 是网络安全领域特定语言，底层融合多种安全能力/工具。
[github: yaklang](https://github.com/yaklang/yaklang)
